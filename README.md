# LUMISAT – CubeSat for Artificial Light Pollution Monitoring

## Project Overview

LUMISAT is a 1.5U CubeSat concept developed for monitoring artificial light pollution using onboard sensing, environmental data acquisition, and real-time telemetry systems.

The project focused on CubeSat subsystem integration, structural design, embedded systems implementation, telemetry development, and payload configuration for environmental monitoring applications.

---

## Mission Objective

The primary objective of LUMISAT was to develop a small satellite platform capable of:

* Monitoring artificial light pollution
* Collecting environmental data
* Demonstrating CubeSat subsystem integration
* Implementing real-time telemetry and visualization
